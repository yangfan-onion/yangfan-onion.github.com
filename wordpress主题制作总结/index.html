wordpress主题制作总结

Aug 07, 2015 in Code
  1. 网站名字 <?php bloginfo('name');?>
  2. 网站描述 <?php bloginfo('description');?>

  3. 查询Post的主循环:

  4. 多语言支持: <?php _e(‘需要翻译的文字’);?> 这里放上_e()为后续的多语言支持提供入口

  5. Post的名字: <?php the_title();?>
  6. Post的链接: <?php the_permalink();?>
  7. Post的缩略内容: <?php the_excerpt();?>
  8. Post的详细内容: <?php the_content();?>
  9. Post的作者: <?php the_author();?>
  10. Post的日期: <?php the_time(‘Y-m-d’);?>
  11. Post的ID: <?php the_ID(); ?>
  12. Post的metadata:
  13. Post下面生成上一页和下一页链接的函数
  14. sidebar的category列表: <?php wp_list_categories($args); ?>
  15. sidebar的页面列表: <?php wp_list_pages( $args ); ?>
  16. sidebar的archive列表: <?php wp_get_archives( $args ); ?>
  17. sidebar的友情链接:
  • 编辑functions.php, 加入如下代码
    add_filter(‘pre_option_link_manager_enabled’,’__return_true’);
  • 后台会出现链接编辑的页面,还可以将链接设为图片，这样就可以用友链的网站logo来表示友链了
  • 在需要显示的地方添加如下代码，args是可选
  1. sidebar的搜索控件: <?php get_search_form( $echo ); ?> $echo=TRUE 表示echo表单字符串，$echo=FALSE表示return表单字符串
  2. sidebar的日历控件: <?php get_calendar( $initial, $echo ); ?>
